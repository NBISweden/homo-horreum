# homo-horreum

## Prerequisites

Python 3 and pip preferably with virtualenv.

To install prerequisites using virtualenv (replace `venv` with whatever you
feel like):

```shell
$ virtualenv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
```


## Initialize database

```shell
$ alembic upgrade head
```
